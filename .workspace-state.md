## 🎯 New Features Added

### 1. Series Circuit Simulator
**Location:** `pages/simulations/series-circuit.vue`
**Component:** `components/SeriesCircuit.vue`

**Features:**
- Interactive series circuit with up to 5 resistors
- Real-time voltage division visualization
- Animated current flow
- Dynamic SVG circuit diagram
- Individual resistor controls (10Ω - 1000Ω)
- Voltage source control (1V - 120V)
- Live calculations: total resistance, current, power
- Voltage drop indicators for each resistor
- Educational content section explaining:
  - Series circuit fundamentals
  - Voltage divider rule
  - Power dissipation calculations
  - Series vs parallel comparison
  - Real-world examples

**Key Concepts:**
- Current is constant through all components
- Voltage divides proportionally to resistance
- Total resistance = sum of all resistances
- Single path failure affects entire circuit

### 2. Voltage Divider Calculator
**Location:** `pages/calculators/voltage-divider.vue`

**Features:**
- Calculate output voltage from input voltage and resistor values
- Interactive SVG circuit diagram with live visualization
- Input voltage control (0 - 1000V)
- R1 (top resistor) control (1Ω - 1MΩ)
- R2 (bottom resistor) control (1Ω - 1MΩ)
- Reverse calculation: find R2 for target V_out
- Real-time results:
  - Output voltage (V_out)
  - Total resistance
  - Circuit current
  - Total power dissipation
  - Voltage divider ratio
- Voltage drop indicators for both resistors
- Educational content covering:
  - Voltage divider fundamentals
  - Key formulas (V_out, I_total, P_total)
  - Common applications (sensors, references, level shifting)
  - Design tips and best practices
  - Loading effects and warnings
  - Real-world design example (12V to 3.3V)

**Key Formula:**
```
V_out = V_in × (R2 ÷ (R1 + R2))
```
